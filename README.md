# Tech Academy API

Backend for Tech Academy internship project.

## Features

* User Register
* User Login
* JWT Authentication
* Course APIs
* Enrollment APIs
* Django Admin Panel

## Tech Stack

* Django
* Django REST Framework
* JWT
* SQLite

## Run Locally

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
